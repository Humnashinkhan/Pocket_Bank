#  Pocket Bank Dashboard

A responsive **Mini Banking Dashboard** built with **React + TypeScript** that allows users to view their balance, track transactions, filter income/expenses, and simulate money transfers with real-time UI updates.

This project was created as part of a frontend assignment focusing on **component structure, state management, validation, and user experience**.
